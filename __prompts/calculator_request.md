# Calculator Request

## User Request
"haz una calculadora simple con aspecto infantil y colores divertido, botones grandes pensado para dispositivo movil"

## Interpretation
- Simple calculator (basic operations: +, -, *, /).
- Child-friendly UI: Pastel/vibrant colors, huge buttons, rounded corners.
- Mobile-first: Full-screen width utilization, easy to tap.
- Design: Playful, bouncy animations, clear numbers.
